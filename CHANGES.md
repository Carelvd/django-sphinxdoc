Changelog for django-sphinxdoc
==============================

## v2.1.0 ‚Äì 2026-01-31:

- [NEW] Support for version control systems (Currently only Git)
- [NEW] Setting: `SPHINXDOC_PROJECT_ROOT`; specifies the common path for projects
- [NEW] Setting: `SPHINXDOC_SOURCE_ROOT`; specifies the common path for uncompiled documentation
- [NEW] Setting: `SPHINXDOC_TARGET_ROOT`; specifies the common path for compiled documentation
- [NEW] Setting: `SPHINXDOC_DOCUMENTATION_DIR`; specifies the fallback directory for the `SPHINXDOC_PROJECT_ROOT`.
- [NEW] Setting: `VERSION_CONTROL_CREDENTIALS`; specifies the necessary server credentials
- [CHANGE] Setting: `SPHINXDOC_BUILD_DIR`; overridden by  `Project.target` and `SPHINXDOC_DOCUMENTATION_TARGET` (or `SPHINXDOC_DOCUMENTATION`)
- [CHANGE] Indices fields now use BigAutoFields (`Project` and `Document`)
- [CHANGE] `Project` added the `root` attribute, renamed `path` to `source` and added `target`.
- [NEW] `Project` added the `repo` attribute to support version control.
- [NEW] `Project` added the `created` and `updated` fields.
- [NEW] `Document` renamed `title` to `name` and `content` to `data`.
